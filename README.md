# To-Do List
Это простое приложение для управления списком задач, реализованное на языке C++

### Клонирование репозитория
```bash
git clone https://github.com/Danil5656/to-do-List
cd to-do-List
```

### Минимальные требования
Компилятор g++ версия 13.3.0 или выше. Проверить версию можно командой:
```bash
g++ -v
```
Если у вас версия ниже, обновите компилятор

Doxygen для генерации документации
```bash
sudo apt install doxygen
```

### Сборка проекта
Для сборки программы в корне проекта выполните команду:
```bash
g++ src/*.cpp -o a.out
```

### Запуск программы
Для запуска программы (после выполнения сборки) выполните команду:
```bash
./a.out
```

### Генерация документации
Для генерации документации в формате html в корне проекта выполните команду:
```bash
doxygen Doxyfile
```
После успешного выполнения откройте файл ./html/index.html в браузере, чтобы посмотреть документацию
